# Foundations Lab Final
Cybersecurity Foundations Lab  
The Knowledge House — Innovation Fellowship

## Overview

This repository documents the final lab completed during the **Cybersecurity Foundations phase of the Innovation Fellowship at The Knowledge House**.

The goal of this lab was to demonstrate the successful setup and verification of a cybersecurity lab environment using virtualization, Linux system validation, and simulated network connectivity.

This repository serves as a **technical artifact archive** containing scripts, logs, and network simulation files that verify the environment was configured correctly.

---

# Lab Objectives

Students were required to demonstrate the following:

- Run a verification script inside an Ubuntu virtual machine
- Confirm system configuration and installed tools
- Demonstrate host and guest systems running simultaneously
- Simulate a network connection using Cisco Packet Tracer
- Capture system verification logs and supporting screenshots

# Artifact Descriptions

## lab_verify.sh

A Bash script designed to verify that the lab environment is correctly configured.

The script checks for:

- Git installation
- Python3 installation
- Network configuration
- Virtualization environment detection
- Disk space availability

### Execution

chmod +x lab_verify.sh
./lab_verify.sh

# Security Concepts Demonstrated

This lab reinforces foundational cybersecurity principles including the **CIA Triad**.

| Principle | Description |
|----------|-------------|
| Confidentiality | Restricting access to sensitive systems and data |
| Integrity | Verifying system configuration through automated scripts |
| Availability | Ensuring the lab environment remains accessible and operational |
